# 🏥 Healthcare System Web Application (MERN Stack)

## 📌 Project Overview

This is a Full Stack Healthcare Web Application built using the MERN stack (MongoDB, Express.js, React, Node.js).
It helps users manage healthcare services like patient records, appointments, and doctor interactions in an efficient and user-friendly way.

---

## 🚀 Features

* 👤 User Authentication (Login / Signup)
* 🩺 Patient Record Management
* 📅 Appointment Booking System
* 👨‍⚕️ Doctor Dashboard
* 📊 Data Handling with Backend API
* 🔐 Secure Data Storage

---

## 🛠️ Tech Stack

**Frontend:**

* React.js (Vite)
* HTML, CSS, JavaScript

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

**Other Tools:**

* Git & GitHub

## ⚙️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/prabhkang17/Healthcare-System-Web-Application-MERN-Stack-.git
cd Healthcare-System-Web-Application-MERN-Stack-
```

---

### 2️⃣ Install Dependencies

👉 For Backend:

```bash
cd backend
npm install
```

👉 For Frontend:

```bash
cd frontend
npm install
```

---

### 3️⃣ Run the Project

👉 Start Backend:

```bash
npm start
```

👉 Start Frontend:

```bash
npm run dev
```

---

### 4️⃣ Open in Browser

```
http://localhost:5173
```
